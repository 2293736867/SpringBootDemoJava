# 1 Spring Boot demo

该仓库是`Spring Boot`的各种`Demo`合集，使用`Java`实现。

# 2 环境

- `OpenJDK 11.0.8`
- `IDEA 2020.2`（请安装`lombok`插件）
- `Manjaro`
- `Maven3`
